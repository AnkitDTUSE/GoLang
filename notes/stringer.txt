One of the most ubiquitous interfaces is Stringer defined by the fmt package.
ubiquitous -> Present everywhere, very common, or found all over the place

type Stringer interface {
    String() string

}

It allows an object to define:
"How should I be represented as a string?"

conceptually Stringer works like this ->

    if value implements fmt.Stringer {
        print(value.String())
    } else {
        print(default representation)
    }
